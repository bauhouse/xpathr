xpathr

xpathr is an open source collaborative XSLT debugging tool developed with Symphony CMS and the GitHub API for sharing code snippets as gists.

If you want to get involved to help make xpathr better (or perhaps fix a bug you've found), please fork xpathr on GitHub and send a pull request.
